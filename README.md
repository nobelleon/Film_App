## Features

- Get movies from TMDb JSON API
- RxDart
- Used BloC design patterns
- Adding Call Demo
- Adding Chat Demo
- Adding Video Call Demo


## Plugins :

Install my-project at pubspec.yaml

```bash
  dio: ^3.0.0
  bloc: ^7.0.0 
  rxdart: ^0.26.0
  eva_icons_flutter: ^3.0.0
  flutter_rating_bar: ^4.0.0
  page_indicator: ^0.4.1
```
